Test repository to learn creating github page.

Following are the steps to do so:

  • Creat an empty repository.
  • Go to settings for the same repo.
  • Go to page layout (at bottom of page).
  • Choose any layout.
  • Publish the page.
  • On next page you will see a url at the top that url is for your repository specific page url.
